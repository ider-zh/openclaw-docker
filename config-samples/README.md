# Config Samples

This directory contains version-specific example configuration files for OpenClaw.

## Structure

- `config-samples/openclaw/` — versioned sample `openclaw.json` files.

## Naming convention

- `vYYYY.M.D.json` — corresponds to OpenClaw version tags like `v2026.5.12`

## Usage

- Copy the relevant file into `openclaw/openclaw.json` when testing a specific version.
- Replace placeholder credentials such as `YOUR_GATEWAY_TOKEN_HERE` and `YOUR_API_KEY_HERE`.
